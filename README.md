<h1>Project Overview</h1>

<h3>This is a Recipe sharing React project with a Node.js backend.</h3>

**Getting Started**

git clone <repository-url>

cd frontend

npm install

cd ../backend

npm install

**To run frontend**

cd frontend

npm run dev

**To run backend**

cd backend

npm run start

Frontend Technologies

React

JavaScript

taiwind

Backend Technologies

Node.js

Express.js

MongoDB
